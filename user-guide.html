User Guide

The User Guide covers all aspects of GeoEq, from basic soil property calculations to advanced geotechnical modeling. Each section introduces a topic (such as "unit weight calculations") and discusses how GeoEq approaches the problem, complete with code examples.

GeoEq is designed to be lightweight, traceable, and reliable. Every formula used in this library is cross-referenced with standard geotechnical textbooks like Das, Holtz, and Craig.

Soil Properties

Soil properties form the foundation of geotechnical engineering. GeoEq provides a comprehensive set of functions to calculate these properties from basic laboratory data.

python 
from geoeq.soil import properties as sp

# Calculate void ratio from porosity
void_ratio = sp.void_ratio_from_porosity(n=0.42)
print(f"Void Ratio: {void_ratio:.3f}")

All property functions validate that input values fall within physically possible ranges (e.g., porosity cannot be greater than 1 or less than 0).

Unit Weight Calculations

Calculating the dry, saturated, and submerged unit weights of soil is a frequent task. GeoEq implements these using specific gravity (Gs), void ratio (e), and moisture content (w).

python 
# Saturated unit weight (kN/m3)
gamma_sat = sp.saturated_unit_weight(Gs=2.65, e=0.68, unit_weight_water=9.81)
print(f"Saturated Unit Weight: {gamma_sat:.2f} kN/m3")

Atterberg Limits

Atterberg limits define the critical water contents of a fine-grained soil. GeoEq calculates the Plasticity Index (PI) and Liquidity Index (LI) used for soil classification.

python 
# Calculate Plasticity Index
pi = sp.plasticity_index(LL=45, PL=20)

# Calculate Liquidity Index
li = sp.liquidity_index(w=32, PL=20, PI=pi)

Behind the Code

GeoEq isn't just a library; it's an engineering tool. We believe in transparency. Below is a summary of the core equations implemented in the current version.

Dry Unit Weight: γd = (Gs γw) / (1 + e)

Porosity to Void Ratio: e = n / (1 - n)
