GeoEq

A Python library for geotechnical engineering calculations.

Install

pip install geoeq

Quick start

import geoeq as ge
from geoeq.soil import properties as sp

gamma_d = sp.dry_unit_weight(Gs=2.65, e=0.72)
print(f"Dry unit weight: {gamma_d:.2f} kN/m3")
